# 🎓 Грейдер-аналитика: ETL-система интеграции с онлайн-университетом #
Проект по разработке ETL-скрипта для сбора, обработки и логирования аналитических данных об успеваемости студентов онлайн-университета, использующих встроенный редактор кода (грейдер).
## 📝 Описание проекта ##
Каждый день тысячи студентов отправляют свои решения задач на проверку. Данные о попытках (код, время, статус, ID студента и задачи) хранятся на стороне удаленного сервиса грейдера. 

Скрипт автоматизирует процесс выгрузки данных через API, проводит их валидацию, структурирует и сохраняет в локальную базу данных PostgreSQL. После анализа эффективности обучения выгружает метрики в GoogleSheets таблицу, при этом отправляя уведомление на почту. Весь процесс работы скрипта логируется в файл, в котором хранятся данные за последние 3 дня.

Ссылка на таблицу: https://docs.google.com/spreadsheets/d/1e-ANzHGEP9uwjbBGUNCXiSRPV-p17ClOm3C0NUlCu6o/edit?gid=0#gid=0
